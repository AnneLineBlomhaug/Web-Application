## Introduction

The WeDeliverTech Management System is a web application developed for efficient management of staff members and delivery drivers. It provides a user-friendly interface for adding, tracking, and managing various activities related to staff and delivery operations.

## Features

- **Staff Management:**
  - Add new staff members.
  - Track clock-in and clock-out times.
  - Monitor expected return times.

- **Delivery Driver Management:**
  - Add new delivery drivers with vehicle details.
  - Track delivery addresses and return times.

- **Real-time Notifications:**
  - Receive toast notifications for late staff members and delivery drivers.

- **Responsive Tables:**
  - Easily view and interact with data through responsive tables.

## Usage

As mentioned above, The WeDeliverTech Management System, is an efficient management dashboard that will now record following:

1. Staff members click in- and out. This have also to be used during lunch breaks.
2. Delivers tracking.

General information about the usage and the page;

The top left corner contains a menu options that might be helpful when it comes to inventory and orders.

This system does calculate and show the duration of every individual absence, and also their expected time of return.

If you have been out of the office longer than expecected, there will be a notification on the screen showing staff picture, name, surname and amount of time of absence. 

The dashboard also contains information that keeps track of current deliveries of orders to customers. This results to have better control of current deliveries, which is located under "Schedule Delivery". The drivers type of vehicle, name, surname, telephone number, delivery address and the time each drivers will return to the office, have to be registered. By clicking the "Add" button, will then register delivery information to the Delivery Board Table. 

On the last section of the web application, there is something called "Delivery Board". This contains that if the return time have passed, and the delivery driver has not yet returned, a notification with the name, surname, telephone, estimated return time and address of current delivery will be displayed. When the delivery returns to the office, the user can then select their row in the Delivery Board and click the button "Clear".

Other relevant information is that the current date and time is updated every second at the very bottom of the page. In case of misunderstandings, the format is: "Day - Month - Year - Hour - Minute - Second". 

The dashboard do overall give an overview of who is currently at the office and those who are out of office.
